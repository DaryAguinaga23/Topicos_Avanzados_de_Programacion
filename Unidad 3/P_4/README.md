# P_4

App de recomendaciones de vestimenta según temporada y clima (Jetpack Compose).

## Cómo abrir y ejecutar
1. Abra Android Studio.
2. File > Open… y seleccione esta carpeta (que contiene `settings.gradle.kts`).
3. Espere a que Gradle sincronice. Si se le solicita, instale SDKs/Plugins faltantes.
4. Seleccione un dispositivo/emulador y pulse Run.

## Funcionalidad
- Seleccione temporada (Primavera, Verano, Otoño, Invierno).
- Seleccione clima (Soleado, Nublado, Lluvioso, Ventoso).
- Ajuste la temperatura con el control deslizante.
- Pulse "Recomendar vestimenta" para ver sugerencias basadas en su selección.
